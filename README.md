# Лабораторные по алгоритмам анализа данных 2025

- `omp_matrix.c` - программа демонстрирует ускорение умножения матриц с помощью **OpenMP**. Она сравнивает время выполнения последовательного алгоритма с параллельной реализацией на CPU и вычисляет итоговое ускорение.

- `cuda_specs.cu` - простая утилита для инициализации среды **CUDA**, которая извлекает и выводит ключевые характеристики всех доступных графических процессоров в системе, включая их имя, объем глобальной памяти.

- `cuda_matrix.cu` - программа реализует базовый алгоритм умножения матриц на графическом процессоре с использованием **CUDA**. Она измеряет время выполнения на GPU, сравнивает его с временем выполнения на CPU и вычисляет ускорение.

- `pi.c` - программа вычисляет приближенное значение числа $\pi$ методом прямоугольников с помощью **OpenMP**, пока разница между последовательными приближениями $\pi$ не станет меньше заданной пользователем точности $\varepsilon$. Измеряется и выводится время, затраченное на параллельные вычисления, а также итоговое количество итераций.

- `sections.c` - программа демонстрирует гибридный подход к параллелизму, используя **OMP Sections** для разделения 4 различных задач (независимых функций) и механизм динамического распределения работы для обработки входных данных. Четыре секции (section) выполняются параллельно, каждая из которых реализует отдельную математическую операцию:

    - Разложение числа на сумму квадратов (наибольших).
    - Вычисление числа Фибоначчи (F(n)).
    - Нахождение n-го простого числа.
    - Вычисление суммы всех делителей числа.
